# Recovered Artifacts

This folder keeps source fragments that were recovered during the Medkit to
Medlife cleanup.

These files are archival only:

- `index-source.html` - recovered HTML source snapshot
- `middleware-source.ts` - recovered middleware snapshot
- `package.json.source.json` - recovered package manifest snapshot
- `package-lock-snippet.json` - recovered lockfile fragment
- `backend-source.py` - recovered backend source that had been mislabeled as audio

Nothing in this folder is used by the production app.
