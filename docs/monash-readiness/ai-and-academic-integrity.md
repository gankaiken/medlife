# AI And Academic Integrity

## AI Roles
- Patient simulator
- Optional feedback assistant
- Assessment assistant only in a formative, reviewable sense

## Human Evidence Preserved
- Learner questions
- Investigations ordered
- Diagnosis and management choices
- Safety decisions
- Learner-authored reflection

## Student Disclosure
- Which activities use AI
- Which outputs are deterministic
- Which feedback is educator reviewed or not reviewed

## Educator Controls
- Disable Text AI Patient Mode
- Disable AI debrief
- Use guided and rule-based modes only

## Failure Modes
- AI unavailability
- Validation failure
- Rule-based fallback
- Evidence receipt or integrity warning

## Formative Use Limitation
- Not for formal progression decisions without additional validation and authorised configuration
