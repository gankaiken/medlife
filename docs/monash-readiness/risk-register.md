# Risk Register

| Risk | Current Mitigation |
| --- | --- |
| Clinical inaccuracy | Case governance, references, reviewer workflow, explicit unapproved status |
| AI hallucination | Validation, refusal boundaries, deterministic fallback |
| Incorrect assessment | Formative labels, educator review, agreement workflow |
| Bias | Explicit AI limitations, reviewer audit, contextual safeguards |
| Accessibility | Non-3D mode, low-bandwidth settings, audit checklist |
| Privacy | Cookie auth, export controls, role-restricted review, consent separation |
| Cybersecurity | Existing security verification suite, readiness docs, server-side role checks |
| Student distress | Psychological safety prebrief and adverse-event reporting expectation |
| Overreliance | Non-claims, formative labels, educator oversight |
| Technical failure | Reliability runner, fallback modes, incident monitoring |
| Faculty workload | Limited pilot scope and staged case pack |
| Curriculum mismatch | Candidate mapping only, explicit academic review required |
